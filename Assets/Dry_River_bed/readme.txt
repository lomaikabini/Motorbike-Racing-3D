This package contains a dry river bed landscape usable as racing track or a first person
action game.
The scene is only basicly decorated.
Collision-Meshes for walls and ground, skydome and all textures are included.
Perfect scene for prototyping and gameplay testings.

Textures-sizes between 1024x1024 to 4096x1024.

The scene is not(!) created with the Unity-Terrain.

Total Polycount about 150.000
